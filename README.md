# 🏠 House Price Prediction

A machine learning project for predicting house prices using regression models.

## 📌 Features

- Data Cleaning
- Feature Preprocessing
- Pipeline & ColumnTransformer
- Model Training
- Hyperparameter Tuning
- Model Evaluation

## 🤖 Models Used

- Random Forest
- Gradient Boosting
- XGBoost

## 📊 Evaluation

The models were evaluated using:

- R² Score
- MAE
- MSE
